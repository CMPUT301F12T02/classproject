Glossary of terms to be used in the application:

Request: A  description of the photo, text, or audio that needs to be uploaded to complete the task.
Request Screen: An activity where a user outlines the request for their task.
Task: An optional assignment that can be completed by an individual or the community.  Contains a request.
Task Screen: An activity where a user can view a task created by them or somebody else, and can fulfill the task.
